MusFy 0.1.23 corrige o login apos a atualizacao e reforca a compatibilidade de usuarios antigos.

- Ajusta o backend para autenticar por login, email ou nome ja gravado.
- Permite criar senha para usuarios antigos que existiam sem credencial local.
- Atualiza a tela de login para enviar identidade de usuario como login, sem exigir email real.
- Melhora a mensagem de falha quando o servico local retorna erro de autenticacao.
- Adiciona teste de smoke do backend cobrindo registro, login por usuario, login por email e recuperacao de usuario legado.
