MusFy 0.1.28 corrige a inicializacao do servico local no Linux e abre caminho para uso no navegador.

- Inicia o backend empacotado com as dependencias locais corretas tambem fora do host de servico do Windows.
- Grava logs do backend em `backend.log` dentro da pasta de dados do app para diagnostico.
- Adiciona atalho "Abrir MusFy no navegador" na tela de login.
- Mantem o frontend servido pelo proprio backend local para uso em browser quando o servico estiver ativo.
- Adiciona testes para impedir regressao no bootstrap Linux e na entrada pelo navegador.
