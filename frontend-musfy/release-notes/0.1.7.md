MusFy 0.1.7 fecha o pacote desktop e mobile com fluxo de YouTube mais direto, configuracoes menos expostas e host local mais resiliente.

- Adiciona a acao de baixar so o video na tela de captura do YouTube.
- Remove da interface de configuracoes os blocos de host interno e canal automatico de update que o usuario final nao precisa ver.
- Faz o host local detectar porta ocupada na instalacao e escolher outra automaticamente, mantendo o desktop sincronizado com a porta real.
- Mantem o pipeline de release reforcado com rebuild limpo do instalador e upload mais resiliente para o GitHub Releases.
