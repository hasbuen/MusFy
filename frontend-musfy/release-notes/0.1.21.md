MusFy 0.1.21 endurece a reproducao e melhora o fluxo de busca no YouTube.

- Corrige a reproducao da biblioteca quando o cache offline nao inicializa no bundle do Electron.
- Mantem o streaming local funcionando mesmo se a consulta offline falhar.
- Adiciona skeletons e estados de retry para previas do YouTube bloqueadas ou ainda carregando.
- Permite remover individualmente badges de buscas recentes para reduzir poluicao visual.
- Adiciona testes automatizados para cache offline, fallback de playback e buscas recentes.
