MusFy 0.1.22 corrige o atualizador para preservar dados do usuario.

- Impede que updates apaguem contas, biblioteca, playlists, audios baixados e preferencias locais.
- Remove a configuracao de limpeza automatica de AppData no instalador.
- Altera o script NSIS para preservar dados gerados pelo usuario em desinstalacoes e atualizacoes.
- Adiciona backup e restauracao automatica de ProgramData/AppData durante update para proteger usuarios vindos de instaladores antigos.
- Adiciona testes automatizados que bloqueiam regressao de limpeza indevida de dados no instalador.
