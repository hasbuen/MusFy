MusFy 0.1.6 reforça a atualização automática e o pipeline de release.

- Recupera a interface e o backend se o `quitAndInstall` falhar, em vez de deixar o app sem bandeja ou sem resposta.
- Mantém o fluxo de correção de textos quebrados aplicado ao updater e aos dados dinâmicos exibidos no desktop.
- Adiciona retentativas no publicador de release do GitHub para lidar melhor com falhas de rede durante upload de assets grandes.
- Confirma uploads já concluídos após erro de rede e volta a tentar somente o que realmente ficou inconsistente no release.
