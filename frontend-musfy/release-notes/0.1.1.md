MusFy 0.1.1 entrega um desktop mais consistente no uso diario.

- Corrige o modo tela cheia do player de video com controles visiveis dentro do proprio painel.
- Mantem mini player, video principal e sincronizacao entre janelas alinhados no desktop.
- Exibe aviso no app quando uma nova atualizacao esta disponivel ou pronta para instalar.
- Mostra notas da versao dentro da interface para o usuario entender o que mudou antes de atualizar.
- Publica releases do Windows no GitHub com `latest.yml`, nome da versao e notas consumidas pelo updater.
