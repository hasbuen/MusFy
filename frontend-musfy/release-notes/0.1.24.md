MusFy 0.1.24 corrige a inicializacao do servico local e mantem os ajustes de login da versao anterior.

- Corrige o instalador para embarcar as dependencias do backend local, incluindo Express.
- Garante que o servico Windows consiga subir a API em `127.0.0.1:3001` apos a atualizacao.
- Mantem autenticacao por login, email ou nome ja gravado.
- Mantem recuperacao de usuarios antigos que existiam sem senha local.
- Adiciona teste para impedir regressao no empacotamento do backend.
